# VJN Way To Success - Billing & Inventory Management System

Comprehensive billing and inventory management system with cross-platform desktop and mobile applications.

## 🚀 Tech Stack

**Backend**
- Node.js + Express.js + TypeScript
- SQLite Database (better-sqlite3)
- JWT Authentication

**Desktop (Electron)**
- React 18 + TypeScript
- Cross-platform: Windows, macOS, Linux

**Mobile (React Native)**
- Expo Framework + TypeScript
- Cross-platform: Android, iOS, Web

**Development**
- VS Code, Git & GitHub, Postman
- TypeScript, esbuild, electron-builder

## 🎯 Core Features

✅ Authentication & Authorization  
✅ Client Management  
✅ Inventory & Stock Tracking  
✅ Order Management & Dispatch  
✅ Invoice Generation (PDF)  
✅ Payment Processing  
✅ Worker Management  
✅ Analytics Dashboard  
✅ Report Generation  
✅ Automated Backup  
✅ AI-Powered Insights  

## 📁 Project Structure

```
├── backend/          # API Server (Node.js + Express + SQLite)
├── desktop/          # Desktop App (Electron + React)
├── frontend/         # Mobile App (React Native + Expo)
└── data/            # Database Files
```
